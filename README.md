# Desk Lighting Controller Project
#### Basic Lighting Controller Project
### Files Included
  - Arduino Files
  - 3d STL Files
### Features
- On Off switch
- Lighting Mode Switch
  - Red
  - Green
  - Blue
  - Transitioning
 
### Needed Parts
- 5v Rated RBG Lighting Strip [Example](https://a.co/d/iguN1RD)
